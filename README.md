# java-cli-bloop-sbt-hsqldb-simple

## Description
A java sbt build, that connects to hsqldb
database.

## Tech stack
- java
- bloop
- sbt
- hsqldb driver

## Docker stack
- eed3si9n/sbt

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
